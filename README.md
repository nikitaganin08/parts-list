# Тестовое приложение для стажировки

#### Используемые технологии:
* Maven;
* Tomcat;
* Spring 5.0.8;
* Hibernate 5.3.6.Final;
* MySQL (база данных). 
    >Название базы `test`, с логином и паролем `root`;
* Frontend: Angular 6;

#### Для запуска:
mvn spring-boot:run

#### В браузере ввести следующее:
http://localhost:8080
